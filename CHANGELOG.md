# Changelog

## 0.1.3 - 2026-05-05

- Reset public release channel for hardened signed builds.
- Public GitHub Releases no longer distribute Python wheels, Python source archives, or npm tarballs.
- Desktop binaries will be attached only after signed release artifacts are available.
- Added a typo-friendly interactive CLI menu so normal chunked commits do not require long flag syntax.
- Interactive CLI uses the current repo automatically and prompts for a path only outside a Git worktree.
- Refined internal architecture and CI checks to improve release quality and maintainability.
